# Análisis de Ventas de Manzanas con NumPy

## Descripción del proyecto

Este proyecto realiza un análisis de una base de datos de ventas de manzanas utilizando Python y la librería NumPy.  
El desarrollo se llevó a cabo en un notebook de Google Colab y los datos están organizados por meses y ciudades.

El objetivo es aplicar operaciones matemáticas, manipulación de arrays, agrupación de columnas y visualización de datos para obtener información relevante sobre las ventas.

---

## Tecnologías utilizadas

- Python 3
- Google Colab
- NumPy
- Matplotlib

## Contenido del análisis

Dentro del notebook se realizan las siguientes tareas:

- Carga de la base de datos de ventas.
- Conversión de los datos a arrays de NumPy.
- Análisis de ventas por meses y ciudades.
- Operaciones matemáticas sobre arrays:
  - Suma de ventas
  - Promedios
  - Aplicación de ecuaciones
- Agrupación de columnas para obtener métricas generales.
- Generación de números aleatorios:
  - Enteros
  - Decimales
- Uso de números aleatorios para simulaciones y pruebas.
- Creación de gráficas para visualizar los resultados.

- Practicar el uso de NumPy para el manejo de datos.
- Aplicar operaciones matemáticas y transformaciones sobre arrays.
- Analizar datos de ventas de forma estructurada.
- Generar y manipular números aleatorios.
- Visualizar información mediante gráficas.
